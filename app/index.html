Hey You

'Hey You' iPhone App

- gGitHub Organization

Hey You is a map-based missed connections app for the iPhone. The app connects with a custom made REST API backend build in Node.js with the Express framework.

My five-person team built the MVP in less than a week. Aside from pitching the project idea, my responsibilities included bulding API routes for serving up JSON data for the iOS app to consume, designing the MongoDB schemas, and creating Mocha and Chai tests for validing functionality on Travis-CI.org.

App Sauce

'App Sauce Co' Android Apps

- gDrinking Time GitHub ProjecthDeveloper Page

Under the name "App Sauce Co," I designed, programmed, and published 6 Android applications in the Google Play Store using Java and the Eclipse IDE. In total, my apps have been downloaded over 70,000 times.

These novelity apps included a Flappy Bird text-based game, several versions of a game paired with TV-shows, and a pig-latin translator. All apps were made with a customized UI from scratch, and most apps had a specialized alternate tablet interface.

Note: I have since unpublished some of these apps from the Google Play Store. However, all apps have been open-sourced under my GitHub name.
