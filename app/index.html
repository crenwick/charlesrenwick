Hey You iPhone application

- gGitHub Organization

A map-based missed connections App built entirely in Objective-C, with a custom REST API backend built in Node.js with the Express framework. The group consisted of two iOS development students, and three Javascript development students and a viable app was built in less than a week.

My responsibilities included building API routes for serving up JSON data for the iOS front end to consume, building JS modules for non-API tasks, and setting up Mocha/Chai tests for validating functionality via continuous integration on Travis-CI.org.
